# Data-Governance-Tool
This project presents a  Metadata-driven Data Governance tool designed to improve visibility, organization, and management of enterprise data assets.

The solution was developed in response to **LGPD (Brazil’s General Data Protection Law)** requirements, enabling organizations to better understand, classify, and govern their data landscape.

## Objectives
- Provide a centralized view of enterprise data assets  
- Enable metadata cataloging and standardization  
- Define data ownership and accountability  
- Ensure data lineage and traceability  
- Improve data quality management  
- Support regulatory compliance (LGPD)  

## Key Features

### Metadata Catalog
- Central repository of data elements and tables  
- Standardized business definitions  
- Data classification and taxonomy  

### Data Governance Framework
- Assignment of **Data Owners** and **Data Stewards**  
- Governance pillars:
  - Accountability  
  - Prioritization  
  - Standardization  
  - Data Lineage  
  - Data Quality  

### Data Lineage
- Traceability of data across systems  
- Visibility into upstream and downstream dependencies  

### Data Quality
- Definition of quality rules  
- Monitoring of critical data elements  

### Search & Discovery
- User-friendly interface for querying metadata  
- Enables both business and technical users to explore data assets
